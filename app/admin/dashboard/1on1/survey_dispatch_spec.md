### [FINAL SPEC] The Signature: 당신의 인력과 잔상 (V13.0)

### 1. 어드민 자동화: [대화 매듭짓기] 팝업 로직
대화 타이머가 종료되는 순간, 운영자가 수동으로 버튼을 찾을 필요 없이 시스템이 즉시 다음 단계를 가이드합니다.

### 1.1. 타이머 종료 트리거 (Auto-Trigger)
* **발생 조건**: `Admin1on1Dashboard` 내 대화 타이머가 `00:00` 도달 시.
* **시스템 동작**: 어드민 화면 중앙에 [회차 확인] 모달 강제 노출.
* 만약 실수로 모달을 종료 했을 시, 모달을 다시 켤 수 있는 버튼 필요함.
* **모달 구성**:
    * **질문**: "방금 종료된 대화는 몇 번째 인연이었나요?"
    * **선택**: `[1회차]`, `[2회차]`, `[3회차]`, `[4회차]`, `[5회차]`
    * **실행**: `[인연의 잔상 발송]` 버튼 클릭 시 해당 회차 유저들에게 실시간 설문 투하.

### 2. 유저 인터렉션: "인연의 잔상 남기기" (The Vibe Check)
대화 직후의 여운을 데이터로 치환하는 3초 컷 피드백 시스템입니다.

### 2.1. 주파수 확인 (The Vibe)
* **문구**: "{상대방}님과의 대화, 어떠셨나요?"
* **옵션**: ⚡️스파크(기대 이상) / ☕️잔잔함(편안함) / 🧊아쉬움(결이 다름)

### 2.2. 매력 포인트 (The Charm)
* **문구**: "상대방의 어떤 점이 가장 기억에 남나요? (최대 2개)"
* **키워드**: 다정다감 / 세심한 배려 / 예쁜 말투 / 매력적 외모 / 깊은 가치관 / 유머러스함 등

### 3. 최종 리포트 핵심 섹션: [The Signature]

### ### #: 0 [Aura Card] 인연의 잔상
* **목적**: 인스타 스토리 공유용 시각화 카드.
* **내용**: 오늘 당신이 발산한 고유한 아우라 컬러와 시그니처 수식어.

### ### #: 1 [The Lone Pioneer] 1/10의 고집
* **내용**: 10명 중 나만 혹은 극소수만 선택한 가치관을 분석하여 '독보적 취향' 증명.
* **카피**: "모두가 유행을 쫓을 때, 당신만이 알아본 [{가치관명}]의 가치."

### ### #: 2 [The Echo] 당신이 남긴 온도
* **내용**: 실제 대화 상대들이 남긴 `Vibe Check` 데이터 기반의 매력 분석.
* **카피**: "당신을 만난 인연들이 기억하는 당신의 '예쁜 말투'는 오늘 최고의 주파수였습니다."

### ### #: 3 [Persona Paradox] 매력적 모순
* **내용**: 내 비딩 의도(내면)와 타인이 느낀 매력(외부)의 간극을 통한 반전 매력 분석.
* **서사**: "성공을 추구하는 당신의 눈빛에서, 타인은 오히려 신뢰할 수 있는 다정함을 발견했습니다."

### ### #: 4 [Subconscious Frequency] 무의식적 주파수
* **내용**: 내가 호감을 보낸(Heart) 이들의 공통 성향 분석을 통한 본능적 이상형 도출.
* **서사**: "이성은 안정을 말하지만, 당신의 본능은 끊임없이 도전하는 이들을 갈구하고 있습니다."

### 4. 기술 및 데이터 명세
* **Admin UI**: `timeLeft === 0` 상태 감지 시 `showModal(FeedbackRoundModal)` 실행.
* **Realtime**: `system_settings` -> `active_feedback_round` 업데이트로 전 유저 동기화.
* **Data Integrity**: 텍스트 잘림 방지를 위해 모든 리포트용 라벨은 **4글자** 이내 단축어 사용.
